# chaos-data

Chaos Viewer atlas data for sm64ds-decomp, published on this orphan branch so the
main history stays clean. Discovered automatically by the viewer at
https://bmanus2-dotcom.github.io/chaos-viewer/ when you enter this repo's link.

Regenerate from a decomp checkout with the ChaosViewer generator, then push the
refreshed chaos-db.json and details/ here.
